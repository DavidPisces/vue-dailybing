# Vue DailyBing
----

An elegant Vue app for daily Bing Wallpaper.

一个非常优雅的每日Bing壁纸Vue App

Just for fun.Enjoy it.

## 项目部署

```sh
npm install
```

### 开发环境部署

```sh
npm run dev
```

### 编译

```sh
npm run build
```
